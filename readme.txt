== Description ==
This plugin is very simple.Reduce the size of the source code by removing the line blake and white space.;Note: Don't use html tag &lt;pre&gt; , please use &lt;div&gt; and &lt;code&gt; and CSS.If the tag does not appear, and decoding is performed
== Installation ==

Upload to WordPress plugins directory /wp-content/plugins
- Activate the plugin in WordPress plugin page
